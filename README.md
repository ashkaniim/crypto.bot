
Developed a fully automated Crypto Signal Bot for Telegram that analyzes market data from multiple exchanges (Binance, CoinMarketCap, etc.) and sends real-time buy/sell alerts to users.
The bot was built in Python using the aiogram library for Telegram integration and ccxt for market data.
Key features:

Real-time data fetching and signal generation

Technical indicator-based strategy (RSI, MACD, Moving Averages)

Configurable alert thresholds and risk filters

Dockerized deployment and webhook support

Logging and user access control for premium channels

Repository: https://github.com/ashkaniim/crypto-signal-bot
